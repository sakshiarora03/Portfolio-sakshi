This is my personal developer portfolio designed to showcase my projects, technical skills, and background as a Computer Science Engineering student.

The portfolio focuses on creating an engaging and modern browsing experience through smooth animations, interactive UI elements, and responsive design.

## Setup & Installation

No build tools or dependencies are required.

### 1 Clone the repository

```bash
git clone https://github.com/rajsvmahendra/portfolio.git
cd portfolio
```

### 2 Open locally

You can simply open the project in your browser.

Option 1:

```
Open index.html directly in your browser
```

Option 2 (recommended):

```bash
npx serve .
```

or

```bash
python -m http.server 8000
```

---

## Deployment

### GitHub Pages

1. Go to **Repository → Settings → Pages**
2. Select **main branch**
3. Save

Your site will be live at:

```
https://rajsvmahendra.github.io/portfolio
```
